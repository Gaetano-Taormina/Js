# Exercise 3

**Italiano:**

crea un contatore che incrementa ad ogni visita della pagina

**English:**

create a counter that increments with each page visit

## Reasoning

**Italiano:**

```text
- ALL'AVVIO della pagina
  - Recupera il valore del contatore dal Local Storage
  - SE il valore non esiste
    - Imposta il contatore a 0
  - Incrementa il contatore di 1
  - Salva il nuovo valore del contatore nel Local Storage
  - Mostra il valore aggiornato nella pagina
```

**English:**

```text
- ON page load
  - Retrieve the counter value from Local Storage
  - IF the value does not exist
    - Set the counter to 0
  - Increment the counter by 1
  - Save the new counter value in Local Storage
  - Display the updated value on the page
```
