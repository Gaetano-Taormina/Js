# Exercise Extra

**Italiano:**

Dato un array di nomi, stampare per ogni nome il prefisso "Sig."

**English:**

Given an array of names, print each name with the prefix "Mr."

## Reasoning

**Italiano:**

```text
* dichiara un array di nomi
* usando forEach() itera su tutti gli elementi:
  * concatena il prefisso al nome
```

**English:**

```text
* declare an array of names
* using forEach() iterate over all elements:
  * append prefix to name
```
