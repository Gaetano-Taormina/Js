# Exercise 1

**Italiano:**

Data una lista di prezzi originali, applica uno sconto del 20% a ciascun prodotto.

**English:**

Given a list of original prices, apply a 20% discount to each product.

## Reasoning

**Italiano:**

```text
* dichiara una funzione applyDiscount
* dichiara un array come lista prezzi
* Usa map() per applicare la funzione:
  * applica lo sconto agli elementi dei prezzi
* stampa il risultato
```

**English:**

```text
* declare a function applyDiscount
* declare an array as list price
* Use map() to apply the function:
  * apply the discount on price's elements
* print the result
```
