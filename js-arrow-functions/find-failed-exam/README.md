# Exercise 2

**Italiano:**

Data una lista di voti di esami, trova il primo esame non superato.

**English:**

Given a list of exam grades, find the first failed exam.

## Reasoning

**Italiano:**

```text
* Dichiara array voti
* Usa find() con arrow function per trovare il primo < 18
```

**English:**

```text
* Declare grades array
* Use find() with arrow function to find the first < 18
```
