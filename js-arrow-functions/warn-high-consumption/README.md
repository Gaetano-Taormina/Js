# Exercise 3

**Italiano:**

Data una lista di consumi elettrici giornalieri in kWh, stampa ogni consumo. Usa console.warn per i consumi alti.

**English:**

Given a list of daily electrical consumptions in kWh, print each consumption. Use console.warn for high consumptions.

## Reasoning

**Italiano:**

```text
* Usa forEach() e arrow function
* Se consumo > soglia:
  * usa console.warn
* Altrimenti:
  * usa console.log
```

**English:**

```text
* Use forEach() and arrow function
* If consumption > threshold:
  * use console.warn
* Otherwise:
  * use console.log
```
