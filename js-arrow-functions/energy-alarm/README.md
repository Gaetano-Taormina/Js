# Esercizio 20 / Exercise 20

**Italian:** data una lista di consumi elettrici giornalieri in kWh, stampa ogni consumo. Usa console.warn per i consumi che superano una certa soglia di allarme

**English:** given a list of daily electricity consumptions in kWh, print each consumption. Use console.warn for consumptions that exceed a certain alarm threshold

## Reasoning

**Italiano:**

```text
- Crea un array di consumi
- Usa forEach() con arrow function:
  - Stampa il consumo
  - Se il consumo > soglia:
    - console.warn()
```

**English:**

```text
- Create consumptions array
- Use forEach() with arrow function:
  - Print consumption
  - If consumption > threshold:
    - console.warn()
```
