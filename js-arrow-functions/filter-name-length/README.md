# Exercise 4

**Italiano:**

Data una lista di nomi, trova solo quelli che hanno una lunghezza tra 3 e 5 caratteri.

**English:**

Given a list of names, find only those that have a length between 3 and 5 characters.

## Reasoning

**Italiano:**

```text
* Usa filter() e arrow function
* Filtra dove length >= 3 e length <= 5
```

**English:**

```text
* Use filter() and arrow function
* Filter where length >= 3 and length <= 5
```
