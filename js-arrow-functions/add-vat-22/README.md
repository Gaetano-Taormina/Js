# Exercise 14

**Italiano:**

Data una lista di prezzi al supermercato, stampa ogni prezzo con l'IVA al 22% inclusa.

**English:**

Given a list of supermarket prices, print each price with 22% VAT included.

## Reasoning

**Italiano:**

```text
* Usa forEach() e arrow function
* Stampa il prezzo moltiplicato per 1.22
```

**English:**

```text
* Use forEach() and arrow function
* Print the price multiplied by 1.22
```
