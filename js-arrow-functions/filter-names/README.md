# Esercizio 21 / Exercise 21

**Italian:** data una lista di nomi, trova solo quelli che hanno una lunghezza tra 3 e 5 caratteri

**English:** given a list of names, find only those that have a length between 3 and 5 characters

## Reasoning

**Italiano:**

```text
- Crea array di nomi
- Usa filter() con arrow function:
  - Ritorna name.length >= 3 AND name.length <= 5
- Stampa nomi filtrati
```

**English:**

```text
- Create names array
- Use filter() with arrow function:
  - Return name.length >= 3 AND name.length <= 5
- Print filtered names
```
