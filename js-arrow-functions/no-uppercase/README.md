# Exercise 5

**Italiano:**

Data una lista di parole, verifica che nessuna abbia un carattere maiuscolo.

**English:**

Given a list of words, verify that none have an uppercase character.

**Italiano:**

```text
- Crea array di parole
- Usa every() con arrow function: parola === parola.toLowerCase()
- Stampa il risultato booleano
```

**English:**

```text
- Create words array
- Use every() with arrow function: word === word.toLowerCase()
- Print boolean result
```
