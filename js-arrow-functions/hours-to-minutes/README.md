# Exercise 8

**Italiano:**

Data una lista di ore lavorate ogni giorno della settimana, stampa per ogni giorno il tempo relativo lavorato, in minuti.

**English:**

Given a list of hours worked each day of the week, print for each day the relative time worked, in minutes.

## Reasoning

**Italiano:**

```text
* Usa forEach() e arrow function
* Stampa l'elemento moltiplicato per 60
```

**English:**

```text
* Use forEach() and arrow function
* Print the element multiplied by 60
```
