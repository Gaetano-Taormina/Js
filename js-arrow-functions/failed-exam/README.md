# Esercizio 22 / Exercise 22

**Italian:** data una lista di voti di esami, trova il primo esame non superato

**English:** given a list of exam grades, find the first failed exam

## Reasoning

**Italiano:**

```text
- Crea array dei voti
- Usa find() con arrow function: voto < 6
- Stampa il primo voto fallito
```

**English:**

```text
- Create grades array
- Use find() with arrow function: grade < 6
- Print the first failed grade
```
