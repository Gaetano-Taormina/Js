# Exercise 16

**Italiano:**

Data una lista di giorni rimanenti alla scadenza di documenti, trova il primo documento che scade entro 7 giorni.

**English:**

Given a list of days remaining until document expiration, find the first document that expires within 7 days.

## Reasoning

**Italiano:**

```text
* Usa find() e arrow function
* Cerca se giorni <= 7
```

**English:**

```text
* Use find() and arrow function
* Search if days <= 7
```
