# Esercizio 19 / Exercise 19

**Italian:** data una lista di prezzi originali, applica uno sconto del 20% a ciascun prodotto

**English:** given a list of original prices, apply a 20% discount to each product

## Reasoning

**Italiano:**

```text
- Crea un array di prezzi
- Usa map() con arrow function: price => price * 0.8
- Stampa i prezzi scontati
```

**English:**

```text
- Create prices array
- Use map() with arrow function: price => price * 0.8
- Print discounted prices
```
