# Esercizio 3

**Italiano:**

Stai creando un contatore di click; incrementa il numero visualizzato nel pulsante dopo ogni click.

**English:**

You are creating a click counter; increment the number displayed in the button after each click.

## Reasoning

**Italiano:**

```text
* Inizializziamo una variabile contatore a 0
* Selezioniamo il pulsante dal DOM
* Aggiungiamo un ascoltatore per l'evento 'click'
* Ad ogni click incrementiamo il contatore e aggiorniamo il testo del pulsante
```

**English:**

```text
* Initialize a counter variable to 0
* Select the button from the DOM
* Add an event listener for the 'click' event
* On each click, increment the counter and update the button's text
```
