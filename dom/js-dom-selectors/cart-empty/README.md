# Esercizio 6

**Italiano:**
Devi svuotare un carrello; cancella l'articolo quando viene cliccato il pulsante corrispondente.

**English:**
You need to empty a cart; delete the item when the corresponding button is clicked.

## Reasoning

**Italiano:**

```text
- Seleziona tutti i pulsanti di eliminazione
- Per ogni pulsante:
  - Aggiungi un evento al click:
    - Rimuovi l'elemento genitore (LI) dal DOM
```

**English:**

```text
- Select all delete buttons
- For each button:
  - Add click event listener:
    - Remove the parent element (LI) from the DOM
```
