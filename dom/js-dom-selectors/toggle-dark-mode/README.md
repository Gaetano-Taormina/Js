# Esercizio 5

**Italiano:**

Vuoi cambiare il tema della pagina; aggiungi la classe .dark-mode quando l'utente clicca sul pulsante del cambio tema.

**English:**

You want to change the page theme; add the .dark-mode class when the user clicks the theme change button.

## Reasoning

**Italiano:**

```text
* Definiamo una classe CSS '.dark-mode' per gli stili scuri
* Selezioniamo il pulsante di cambio tema dal DOM
* Aggiungiamo un ascoltatore per l'evento 'click' al pulsante
* Utilizziamo 'classList.toggle' per aggiungere o rimuovere la classe 'dark-mode' dal body
```

**English:**

```text
* Define a '.dark-mode' CSS class for dark styles
* Select the theme toggle button from the DOM
* Add an event listener for the 'click' event to the button
* Use 'classList.toggle' to add or remove the 'dark-mode' class from the body
```
