# Esercizio 10

**Italiano:**
Un utente sta scrivendo un commento; conta i caratteri rimanenti in tempo reale mentre scrive.

**English:**
A user is writing a comment; count the remaining characters in real time as they type.

## Reasoning

**Italiano:**

```text
- Seleziona la textarea e il contatore
- Al cambio del testo:
  - Calcola max caratteri - caratteri attuali
  - Aggiorna il testo del contatore col risultato
```

**English:**

```text
- Select textarea and counter element
- On input event:
  - Calculate max chars - current char length
  - Update counter text with the result
```
