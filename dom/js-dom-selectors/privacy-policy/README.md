# Esercizio 8

**Italiano:**
In un form, abilita il tasto "Registrati" solo se il checkbox della privacy è selezionato.

**English:**
In a form, enable the "Register" button only if the privacy checkbox is selected.

## Reasoning

**Italiano:**

```text
- Seleziona il checkbox e il pulsante
- Aggiungi un evento al cambio del checkbox:
  - Imposta lo stato disabilitato del pulsante in base a checkbox non spuntato
```

**English:**

```text
- Select checkbox and button
- Add change event to checkbox:
  - Set disabled state of button based on checkbox un-checked status
```
