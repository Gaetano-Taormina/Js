# Esercizio 9

**Italiano:**
Devi evidenziare una riga di una tabella al passaggio del mouse.

**English:**
You need to highlight a table row on hover.

## Reasoning

**Italiano:**

```text
- Seleziona tutte le righe della tabella (TR)
- Per ogni riga:
  - Al mouse enter:
    - Aggiungi la classe di evidenziazione
  - Al mouse leave:
    - Rimuovi la classe
```

**English:**

```text
- Select all table rows (TR)
- For each row:
  - On mouse enter:
    - Add highlight class
  - On mouse leave:
    - Remove class
```
