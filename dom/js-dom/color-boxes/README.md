# Exercise 8

**Italiano:**

data una lista di colori, genera in pagina una serie di box, ognuno con il suo colore

**English:**

Given a list of colors, generate a series of boxes on the page, each with its own color.

## Reasoning

**Italiano:**

```text
- Seleziona il contenitore vuoto
- Per ogni colore nell'array:
  - Crea un elemento div
  - Imposta lo stile del colore di sfondo
  - Aggiungi il div al contenitore
```

**English:**

```text
- Select the empty container
- For each color in the array:
  - Create a div element
  - Set its background color style
  - Append the div to the container
```
