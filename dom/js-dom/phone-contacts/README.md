# Exercise 10

**Italiano:**

data una lista di contatti telefonici, genera una lista HTML. Al click su un elemento, mostra il numero di telefono corrispondente tramite un alert

**English:**

Given a list of phone contacts, generate an HTML list. On clicking an element, show the corresponding phone number via an alert.

## Reasoning

**Italiano:**

```text
- Seleziona la lista HTML
- Per ogni contatto:
  - Crea un LI col nome del contatto
  - Aggiungi un event listener per il click:
    - Mostra un alert col numero di telefono
  - Aggiungi LI alla lista
```

**English:**

```text
- Select the HTML list
- For each contact:
  - Create an LI with the contact name
  - Add click event listener:
    - Show alert with phone number
  - Append LI to the list
```
