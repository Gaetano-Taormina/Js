# Exercise 4

**Italiano:**

con due input password, mostra un messaggio di errore rosso se i valori non coincidono mentre l'utente digita nel secondo campo

**English:**

With two password inputs, show a red error message if the values do not match while the user types in the second field.

## Reasoning

**Italiano:**

```text
* Selezioniamo i due input e l'elemento per il messaggio di errore dal DOM
* Aggiungiamo un ascoltatore per l'evento 'input' sul secondo campo password
* Confrontiamo i valori dei due campi: se sono diversi, mostriamo l'errore
```

**English:**

```text
* Select the two inputs and the error message element from the DOM
* Add an event listener for the 'input' event on the second password field
* Compare the values of the two fields: if they are different, show the error
```
