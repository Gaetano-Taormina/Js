# Exercise 9

**Italiano:**

dato il menù di una pizzeria, genera dinamicamente una lista HTML che mostri nome e prezzo di ogni pizza

**English:**

Given a pizzeria menu, dynamically generate an HTML list showing the name and price of each pizza.

## Reasoning

**Italiano:**

```text
- Seleziona l'elemento lista UL
- Per ogni pizza nel menu:
  - Crea un tag LI
  - Imposta il testo col nome della pizza e il prezzo
  - Aggiungilo alla lista
```

**English:**

```text
- Select the UL list element
- For each pizza in the menu:
  - Create an LI tag
  - Set text with pizza name and price
  - Append it to the list
```
