# Exercise 7

**Italiano:**

dato un oggetto prodotto con una quantità, mostra un badge del colore appropriato e con una descrizione appropriata

**English:**

Given a product object with a quantity, show a badge of the appropriate color and with an appropriate description.

## Reasoning

**Italiano:**

```text
- Seleziona l'elemento badge
- Se la quantita del prodotto > 0:
  - Imposta il testo 'In Stock' e la classe success
- Altrimenti:
  - Imposta il testo 'Out of Stock' e la classe danger
```

**English:**

```text
- Select the badge element
- If product quantity > 0:
  - Set text 'In Stock' and success class
- Else:
  - Set text 'Out of Stock' and danger class
```
