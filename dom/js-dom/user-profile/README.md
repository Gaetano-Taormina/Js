# Exercise 6

**Italiano:**

dato un oggetto utente (nome, ruolo, img) e una struttura HTML vuota, inserisci i dati dell'oggetto nei rispettivi elementi della pagina

**English:**

Given a user object (name, role, img) and an empty HTML structure, insert the object's data into the respective page elements.

## Reasoning

**Italiano:**

```text
- Seleziona l'elemento nome, ruolo e immagine
- Assegna obj.name al testo del nome
- Assegna obj.role al testo del ruolo
- Assegna obj.img all'attributo src dell'immagine
```

**English:**

```text
- Select the name, role, and image elements
- Assign obj.name to the name text
- Assign obj.role to the role text
- Assign obj.img to the image src attribute
```
