# Exercise 5

**Italiano:**

dato un prezzo fisso e un input per la quantità, aggiorna in tempo reale un paragrafo che mostra il totale

**English:**

Given a fixed price and an input for quantity, update a paragraph in real time showing the total.

## Reasoning

**Italiano:**

```text
* Definiamo una costante per il prezzo fisso
* Selezioniamo l'input della quantità e l'elemento per mostrare il totale
* Aggiungiamo un ascoltatore per l'evento 'input' sulla quantità
* Calcoliamo il totale moltiplicando il prezzo per la quantità e aggiorniamo il testo
```

**English:**

```text
* Define a constant for the fixed price
* Select the quantity input and the element to show the total
* Add an event listener for the 'input' event on the quantity
* Calculate the total by multiplying the price by the quantity and update the text
```
