# Exercise 3

**Italiano:**

* Chiedi un numero all'utente. Stampa se è pari o dispari.

**English:**

* Ask the user for a number. Print if it is even or odd.

## Reasoning

**Italiano:**

* chiedi un numero all'utente
* leggi numero
* if pari stampa pari
  * else stampa dispari

**English:**

* ask the user for a number
* read number
* if even print even
  * else print odd
