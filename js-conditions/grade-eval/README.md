# Exercise 9

**Italiano:**

* Chiedi un voto da 1 a 10. Stampa "Insufficiente" se < 6, "Sufficiente" se tra 6 e 7, "Buono" se tra 8 e 9, "Ottimo" se 10.

**English:**

* Ask for a grade from 1 to 10. Print "Insufficient" if < 6, "Sufficient" if between 6 and 7, "Good" if between 8 and 9, "Excellent" if 10.

## Reasoning

**Italiano:**

```text
* leggi il voto
* se voto < 6:
  * stampa insufficiente
* altrimenti se voto <= 7:
  * stampa sufficiente
* altrimenti se voto <= 9:
  * stampa buono
* altrimenti:
  * stampa ottimo
```

**English:**

```text
* read the grade
* if grade < 6:
  * print insufficient
* otherwise if grade <= 7:
  * print sufficient
* otherwise if grade <= 9:
  * print good
* otherwise:
  * print excellent
```
