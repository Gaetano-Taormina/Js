# Exercise 12

**Italiano:**

Simula il lancio di una moneta. Se il numero è minore di 0.5 stampa "Testa", altrimenti stampa "Croce".

**English:**

Simulate a coin flip. If the number is less than 0.5 print "Heads", otherwise print "Tails".

## Reasoning

**Italiano:**

```text
* Simula il lancio con `Math.random()`
* Se il numero generato è minore di 0.5:
  * stampa "Testa"
* Altrimenti:
  * stampa "Croce"
```

**English:**

```text
* Simulate the flip with `Math.random()`
* If the generated number is less than 0.5:
  * print "Heads"
* Otherwise:
  * print "Tails"
```
