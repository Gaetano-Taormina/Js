# Exercise 1

**Italiano:**

* Chiedi l'età dell'utente tramite prompt. Se l'utente è maggiorenne, stampa a console "Accesso consentito al locale".

**English:**

* Ask the user's age via prompt. If the user is an adult, print "Access granted to the club" to the console.

## Reasoning

**Italiano:**

```text
* leggi l'età dell'utente
* se l'età è maggiore o uguale a 18:
  * stampa "Accesso consentito al locale"
* altrimenti:
  * stampa "Accesso negato"
```

**English:**

```text
* read the user's age
* if age is greater or equal to 18:
  * print "Access granted to the club"
* otherwise:
  * print "Access denied"
```
