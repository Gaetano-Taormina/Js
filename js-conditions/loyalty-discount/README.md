# Exercise 6

**Italiano:**

* Chiedi se l'utente ha la tessera fedeltà. Se ha la tessera applica uno sconto del 10% al prezzo inserito, altrimenti lascia il prezzo intero. Stampa il prezzo.

**English:**

* Ask if the user has a loyalty card. If they have the card apply a 10% discount to the entered price, otherwise leave the full price. Print the price.

## Reasoning

**Italiano:**

```text
* leggi se l'utente ha la tessera
* leggi il prezzo
* converti il prezzo in numero
* se la risposta per la tessera è affermativa:
  * calcola il prezzo scontato del 10%
* stampa il prezzo finale
```

**English:**

```text
* read if the user has a loyalty card
* read the price
* convert price to number
* if the loyalty card answer is affirmative:
  * calculate the price with a 10% discount
* print the final price
```
