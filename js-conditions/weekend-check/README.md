# Exercise 7

**Italiano:**

* Chiedi all'utente che giorno è oggi. Se è "sabato" o "domenica" stampa "Buon weekend!", altrimenti stampa "Buon lavoro".

**English:**

* Ask the user what day it is today. If it is "saturday" or "sunday" print "Have a good weekend!", otherwise print "Have a good day at work".

## Reasoning

**Italiano:**

```text
* leggi il giorno
* se è "saturday" o "sunday" o le loro varianti maiuscole:
  * stampa "Buon weekend"
* altrimenti:
  * stampa "Buon lavoro"
```

**English:**

```text
* read the day string
* if it is "saturday" or "sunday" or their capitalized variants:
  * print "Have a good weekend"
* otherwise:
  * print "Have a good day at work"
```
