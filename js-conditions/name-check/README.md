# Exercise 13

**Italiano:**

* Gaetano:
Chiedi all'utente di inserire il suo nome. Verifica che non abbia lasciato il campo vuoto.

**English:**

* Gaetano:
Ask the user to enter their name. Verify that they have not left the field empty.

## Reasoning

**Italiano:**

```text
* leggi il nome dell'utente
* se il nome è vuoto:
  * stampa che non è stato inserito nessun nome
* altrimenti:
  * stampa il nome dell'utente
```

**English:**

```text
* read the user's name
* if the name is empty:
  * print that no name has been entered
* otherwise:
  * print the user's name
```
