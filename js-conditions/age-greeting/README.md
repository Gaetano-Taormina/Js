# Exercise 5

**Italiano:**

* Chiedi l'età all'utente. Se è minorenne stampa "Ciao", se è over60 stampa "Salve", altrimenti stampa "Benvenuto".

**English:**

* Ask the user's age. If minor print "Hi", if over 60 print "Greetings", otherwise print "Welcome".

## Reasoning
