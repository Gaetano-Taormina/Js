# Exercise 7

**Italiano:**

Chiedi un numero N all'utente. Calcola la somma di tutti i numeri da 1 a N e stampa il risultato.

**English:**

Ask the user for a number N. Calculate the sum of all numbers from 1 to N and print the result.

## Reasoning

**Italiano:**

```text
* Chiedi N all'utente
* Inizializza somma a 0
* Ciclo for da 1 a N:
  * Aggiungi l'indice alla somma
* Stampa la somma
```

**English:**

```text
* Ask N from the user
* Initialize sum to 0
* For loop from 1 to N:
  * Add the index to the sum
* Print the sum
```
