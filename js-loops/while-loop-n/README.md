# Exercise 3

**Italiano:**

Chiedi un numero N all'utente. Stampa i numeri da 0 a N usando un ciclo while.

**English:**

Ask the user for a number N. Print the numbers from 0 to N using a while loop.

## Reasoning

**Italiano:**

```text
* Chiedi il numero all'utente e salvalo in N
* Inizializza contatore i a 0
* Ciclo while (finché i <= N):
  * Stampa i
  * Incrementa i
```

**English:**

```text
* Ask the user for the number and save it in N
* Initialize a counter i to 0
* While loop (as long as i <= N):
  * Print i
  * Increment i
```
