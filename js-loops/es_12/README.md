# Esercizio 12 / Exercise 12

**Italian:** Il computer pensa a un numero tra 1 e 100. L'utente indovina. Dopo ogni tentativo sbagliato, il computer dice "Troppo alto" o "Troppo basso" per aiutare l'utente.

**English:** The computer thinks of a number between 1 and 100. The user guesses. After each wrong attempt, the computer says "Too high" or "Too low" to help the user.
