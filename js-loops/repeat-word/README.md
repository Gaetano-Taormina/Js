# Exercise 6

**Italiano:**

Chiedi all'utente una parola e un numero N. Stampa quella parola N volte.

**English:**

Ask the user for a word and a number N. Print that word N times.

## Reasoning

**Italiano:**

```text
* Chiedi parola e numero
* Ciclo for da 0 a N:
  * Stampa la parola
```

**English:**

```text
* Ask for word and number
* For loop from 0 to N:
  * Print the word
```
