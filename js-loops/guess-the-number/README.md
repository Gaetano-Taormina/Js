# Esercizio 12

**Italiano:**

Il computer pensa a un numero tra 1 e 100. L'utente indovina. Dopo ogni tentativo sbagliato, il computer dice "Troppo alto" o "Troppo basso" per aiutare l'utente.

**English:**

The computer thinks of a number between 1 and 100. The user guesses. After each wrong attempt, the computer says "Too high" or "Too low" to help the user.

## Reasoning

**Italiano:**

```text
* Generiamo un numero casuale tra 1 e 100 e lo salviamo in una variabile target
* Utilizziamo un ciclo while che continua finché l'utente non indovina il numero
* Ad ogni iterazione, chiediamo all'utente di inserire un numero usando 'prompt'
* Verifichiamo se il numero inserito è minore, maggiore o uguale al target e mostriamo il messaggio corrispondente con 'alert'
```

**English:**

```text
* Generate a random number between 1 and 100 and store it in a target variable
* Use a while loop that continues until the user guesses the number
* At each iteration, prompt the user to input a number using 'prompt'
* Check if the guessed number is lower, higher, or equal to the target and show the corresponding message using 'alert'
```
