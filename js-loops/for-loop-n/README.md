# Exercise 4

**Italiano:**

Chiedi un numero N all'utente. Stampa i numeri da 0 a N usando un ciclo for.

**English:**

Ask the user for a number N. Print the numbers from 0 to N using a for loop.

## Reasoning

**Italiano:**

```text
* Chiedi il numero all'utente e salvalo in N
* Ciclo for con i da 0 a N:
  * Stampa i ad ogni iterazione
```

**English:**

```text
* Ask the user for the number and save it in N
* For loop with i from 0 to N:
  * Print i at each iteration
```
