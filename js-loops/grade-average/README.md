# Esercizio 10 / Exercise 10

**Italian:** Chiedi all'utente di inserire i suoi voti. Stampa la media.

**English:** Ask the user to enter their grades. Print the average.

## Reasoning

**Italiano:**

```text
- Crea un array vuoto per i voti
- Ciclo infinito:
  - Chiedi il voto
  - Se l'input è vuoto, interrompi il ciclo (break)
  - Converti in numero e aggiungi all'array
- Se ci sono voti:
  - Calcola la somma con un for
  - Stampa la media (somma / lunghezza)
```

**English:**

```text
- Create an empty array for grades
- Infinite loop:
  - Prompt for grade
  - If input is empty, break
  - Convert to number and push to array
- If grades exist:
  - Calculate sum with a for loop
  - Print average (sum / length)
```
