# Exercise 9

**Italiano:**

descrivi le informazioni di un evento in calendario

**English:**

describe the information of a calendar event

## Reasoning

**Italiano:**

```text
* Dichiara una costante `calendarEvent` e assegnale un oggetto
* Inserisci le informazioni dell'evento
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `calendarEvent` constant and assign an object to it
* Insert the event information
* Print the object to the console
```
