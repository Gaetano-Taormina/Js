# Exercise 8

**Italiano:**

descrivi i dati di una carta di credito

**English:**

describe the data of a credit card

## Reasoning

**Italiano:**

```text
* Dichiara costante `creditCard` e assegnale un oggetto
* Inserisci i dati della carta
* Stampa l'oggetto in console
```

**English:**

```text
* Declare `creditCard` constant and assign an object to it
* Insert the card data
* Print the object to the console
```
