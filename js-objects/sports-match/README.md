# Exercise 6

**Italiano:**

descrivi lo stato attuale di una partita sportiva

**English:**

describe the current status of a sports match

## Reasoning

**Italiano:**

```text
* Dichiara costante `match` e assegnale un oggetto
* Inserisci lo stato della partita
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `match` constant and assign an object to it
* Insert the match status
* Print the object to the console
```
