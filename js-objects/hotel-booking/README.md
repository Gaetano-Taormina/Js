# Exercise 19

**Italiano:**

descrivi una prenotazione alberghiera con dettagli dell'ospite e servizi richiesti

**English:**

describe a hotel booking with guest details and requested services

## Reasoning

**Italiano:**

```text
- Definisci oggetto prenotazione:
  - guest: stringa
  - room: numero
  - services: array di stringhe
- Stampa oggetto
```

**English:**

```text
- Define booking object:
  - guest: string
  - room: number
  - services: array of strings
- Print object
```
