# Exercise 2

**Italiano:**

descrivi i dati anagrafici di una persona fisica

**English:**

describe the personal data of a physical person

## Reasoning

**Italiano:**

```text
* Dichiara una costante `person` e assegnale un oggetto
* Inserisci le proprietà anagrafiche
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `person` constant and assign an object to it
* Insert the personal properties
* Print the object to the console
```
