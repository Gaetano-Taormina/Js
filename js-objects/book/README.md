# Exercise 4

**Italiano:**

descrivi le informazioni essenziali di un libro

**English:**

describe the essential information of a book

## Reasoning

**Italiano:**

```text
* Dichiara una costante `book` e assegnale un oggetto
* Inserisci le informazioni del libro
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `book` constant and assign an object to it
* Insert the book information
* Print the object to the console
```
