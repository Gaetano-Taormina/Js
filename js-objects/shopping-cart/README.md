# Exercise 11

**Italiano:**

descrivi il carrello di un ecommerce

**English:**

describe the shopping cart of an ecommerce

## Reasoning

**Italiano:**

```text
- Definisci oggetto carrello:
  - user: stringa
  - items: array di oggetti prodotto (id, nome, prezzo)
  - total: numero somma
- Stampa oggetto
```

**English:**

```text
- Define cart object:
  - user: string
  - items: array of product objects (id, name, price)
  - total: number sum
- Print object
```
