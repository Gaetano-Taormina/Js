# Exercise 18

**Italiano:**

descrivi la rosa di una squadra di calcio divisa per ruoli

**English:**

describe the squad of a football team divided by roles

## Reasoning

**Italiano:**

```text
- Definisci oggetto squadra:
  - name: stringa
  - players: oggetto coi ruoli (portieri, difensori, attaccanti array)
- Stampa oggetto
```

**English:**

```text
- Define team object:
  - name: string
  - players: object with roles (goalies, defenders, attackers arrays)
- Print object
```
