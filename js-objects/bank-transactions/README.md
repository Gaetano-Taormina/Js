# Exercise 13

**Italiano:**

descrivi le transazioni bancarie di un conto corrente

**English:**

describe the bank transactions of a checking account

## Reasoning

**Italiano:**

```text
- Definisci oggetto conto:
  - owner: stringa
  - balance: numero
  - transactions: array di movimenti (tipo, importo)
- Stampa oggetto
```

**English:**

```text
- Define account object:
  - owner: string
  - balance: number
  - transactions: array of movements (type, amount)
- Print object
```
