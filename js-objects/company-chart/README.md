# Exercise 12

**Italiano:**

descrivi l'organigramma di un'azienda

**English:**

describe the organizational chart of a company

## Reasoning

**Italiano:**

```text
- Definisci oggetto azienda:
  - ceo: stringa
  - departments: array di dipartimenti (nome, manager)
- Stampa oggetto
```

**English:**

```text
- Define company object:
  - ceo: string
  - departments: array of departments (name, manager)
- Print object
```
