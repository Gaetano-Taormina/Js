# Exercise 10

**Italiano:**

descrivi le specifiche di un corso di formazione

**English:**

describe the specifications of a training course

## Reasoning

**Italiano:**

```text
* Dichiara costante `course` e assegnale un oggetto
* Inserisci le specifiche del corso
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `course` constant and assign an object to it
* Insert the course specifications
* Print the object to the console
```
