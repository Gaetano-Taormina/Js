# Exercise 7

**Italiano:**

descrivi le caratteristiche di un immobile in vendita

**English:**

describe the characteristics of a property for sale

## Reasoning

**Italiano:**

```text
* Dichiara costante `property` e assegnale un oggetto
* Inserisci le caratteristiche dell'immobile
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `property` constant and assign an object to it
* Insert the property characteristics
* Print the object to the console
```
