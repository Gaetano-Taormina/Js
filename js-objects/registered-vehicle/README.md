# Exercise 5

**Italiano:**

descrivi i dati identificativi di un veicolo immatricolato

**English:**

describe the identification data of a registered vehicle

## Reasoning

**Italiano:**

```text
* Dichiara costante `vehicle` e assegnale un oggetto
* Inserisci i dati identificativi
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `vehicle` constant and assign an object to it
* Insert the identification data
* Print the object to the console
```
