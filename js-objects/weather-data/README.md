# Exercise 16

**Italiano:**

descrivi i dati meteorologici organizzati per città e giorni della settimana

**English:**

describe the meteorological data organized by city and days of the week

## Reasoning

**Italiano:**

```text
- Definisci oggetto meteo:
  - city: stringa
  - forecast: array di previsioni (giorno, temp)
- Stampa oggetto
```

**English:**

```text
- Define weather object:
  - city: string
  - forecast: array of predictions (day, temp)
- Print object
```
