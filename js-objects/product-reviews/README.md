# Exercise 14

**Italiano:**

descrivi le recensioni dagli utenti su un prodotto Amazon

**English:**

describe user reviews on an Amazon product

## Reasoning

**Italiano:**

```text
- Definisci oggetto prodotto:
  - id: numero
  - name: stringa
  - reviews: array di recensioni (utente, voto, commento)
- Stampa oggetto
```

**English:**

```text
- Define product object:
  - id: number
  - name: string
  - reviews: array of reviews (user, rating, comment)
- Print object
```
