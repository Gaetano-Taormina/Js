# Exercise 15

**Italiano:**

descrivi la lista degli iscritti ad una newsletter

**English:**

describe the list of subscribers to a newsletter

## Reasoning

**Italiano:**

```text
- Definisci oggetto newsletter:
  - name: stringa
  - subscribers: array di iscritti (email, attivo)
- Stampa oggetto
```

**English:**

```text
- Define newsletter object:
  - name: string
  - subscribers: array of subscribers (email, active)
- Print object
```
