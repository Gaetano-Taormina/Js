# Exercise 17

**Italiano:**

descrivi il catalogo di un negozio diviso per categorie e sottocategorie

**English:**

describe the catalog of a store divided by categories and subcategories

## Reasoning

**Italiano:**

```text
- Definisci oggetto catalogo:
  - categories: array di categorie (nome, sottocategorie array)
- Stampa oggetto
```

**English:**

```text
- Define catalog object:
  - categories: array of categories (name, sub array)
- Print object
```
