# Exercise 20

**Italiano:**

descrivi un itinerario di viaggio giorno per giorno

**English:**

describe a day-by-day travel itinerary

## Reasoning

**Italiano:**

```text
- Definisci oggetto itinerario:
  - destination: stringa
  - days: array di giorni (giorno, attivita)
- Stampa oggetto
```

**English:**

```text
- Define itinerary object:
  - destination: string
  - days: array of days (day, activity)
- Print object
```
