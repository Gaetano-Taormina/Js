# Exercise 1

**Italiano:**

descrivi le proprietà geometriche di un triangolo

**English:**

describe the geometric properties of a triangle

## Reasoning

**Italiano:**

```text
* Dichiara una costante `triangle` e assegnale un oggetto
* Inserisci le proprietà base di un triangolo
* Stampa l'oggetto in console
```

**English:**

```text
* Declare a `triangle` constant and assign an object to it
* Insert the basic properties of a triangle
* Print the object in the console
```
