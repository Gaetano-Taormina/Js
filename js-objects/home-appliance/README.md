# Exercise 3

**Italiano:**

descrivi le specifiche tecniche di un elettrodomestico

**English:**

describe the technical specifications of a home appliance

## Reasoning

**Italiano:**

```text
* Dichiara una costante `appliance` e assegnale un oggetto
* Inserisci le specifiche tecniche
* Stampa l'oggetto in console
```

**English:**

```text
* Declare an `appliance` constant and assign an object to it
* Insert the technical specifications
* Print the object to the console
```
