# Exercise 6

**Italiano:**

mostra l'orario corrente aggiornandolo ogni secondo

**English:**

show the current time, updating it every second

## Reasoning

**Italiano:**

```text
- ALL'AVVIO della pagina
  - Definisci una funzione che calcola l'orario corrente e lo mostra
  - Esegui subito la funzione per mostrare l'orario iniziale
  - Avvia un timer ripetitivo ogni 1 secondo (setInterval)
  - AD OGNI intervallo
    - Esegui la funzione per aggiornare l'orario mostrato
```

**English:**

```text
- ON page load
  - Define a function that calculates the current time and displays it
  - Execute the function immediately to show the initial time
  - Start a repetitive timer every 1 second (setInterval)
  - AT EACH interval
    - Execute the function to update the displayed time
```
