# Exercise 10

**Italiano:**

cambia automaticamente il colore di sfondo della pagina ad intervalli regolari

**English:**

automatically change the background color of the page at regular intervals

## Reasoning

**Italiano:**

```text
- ALL'AVVIO della pagina
  - Definisci una lista di colori o una funzione per generare colori casuali
  - Avvia un timer ripetitivo ad intervalli regolari (setInterval)
  - AD OGNI intervallo
    - Seleziona un nuovo colore
    - Applica il nuovo colore allo sfondo della pagina
```

**English:**

```text
- ON page load
  - Define a list of colors or a function to generate random colors
  - Start a repetitive timer at regular intervals (setInterval)
  - AT EACH interval
    - Select a new color
    - Apply the new color to the page background
```
