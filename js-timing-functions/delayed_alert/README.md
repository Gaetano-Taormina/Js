# Exercise 1

**Italiano:**

mostra un alert 3 secondi dopo il caricamento della pagina

**English:**

show an alert 3 seconds after the page loads

## Reasoning

**Italiano:**

```text
- ALL'AVVIO della pagina
  - Avvia un timer di 3 secondi (setTimeout)
  - QUANDO il timer scade
    - Mostra un alert all'utente
```

**English:**

```text
- ON page load
  - Start a 3-second timer (setTimeout)
  - WHEN the timer expires
    - Show an alert to the user
```
