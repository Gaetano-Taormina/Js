# Exercise 9

**Italiano:**

mostra un nuovo numero casuale ogni X secondi

**English:**

show a new random number every X seconds

## Reasoning

**Italiano:**

```text
- ALL'AVVIO della pagina
  - Definisci una funzione che genera un numero casuale e lo mostra
  - Avvia un timer ripetitivo ogni X secondi (setInterval)
  - AD OGNI intervallo
    - Esegui la funzione per generare e mostrare un nuovo numero casuale
```

**English:**

```text
- ON page load
  - Define a function that generates a random number and displays it
  - Start a repetitive timer every X seconds (setInterval)
  - AT EACH interval
    - Execute the function to generate and display a new random number
```
