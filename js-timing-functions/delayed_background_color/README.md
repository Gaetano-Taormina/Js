# Exercise 2

**Italiano:**

al click su un elemento, cambia il colore dello sfondo dopo un ritardo

**English:**

on click of an element, change the background color after a delay

## Reasoning

**Italiano:**

```text
- AL CLICK sull'elemento bersaglio
  - Avvia un timer con un ritardo specificato (setTimeout)
  - QUANDO il timer scade
    - Cambia il colore di sfondo della pagina o dell'elemento
```

**English:**

```text
- ON CLICK of the target element
  - Start a timer with a specified delay (setTimeout)
  - WHEN the timer expires
    - Change the background color of the page or element
```
