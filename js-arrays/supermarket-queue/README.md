# Exercise 17

**Italiano:**

Alla cassa del supermercato, il primo cliente della fila viene servito e se ne va; rimuovilo della coda per far avanzare gli altri.

**English:**

At the supermarket checkout, the first customer in line is served and leaves; remove them from the queue to move the others forward.

## Reasoning

**Italiano:**

```text
* Dichiara array coda
* Usa shift() per rimuovere il primo elemento
* Stampa la nuova coda
```

**English:**

```text
* Declare queue array
* Use shift() to remove the first element
* Print the new queue
```
