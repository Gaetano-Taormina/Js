# Esercizio 30 / Exercise 30

**Italian:** Dopo una partita di tris, inizializza la griglia di gioco con il valore "vuoto" a tutte le celle.

**English:** After a game of tic-tac-toe, initialize the game grid with the "empty" value for all cells.

## Reasoning

**Italiano:**

```text
- Inizializza un array di lunghezza 9 vuoto
- Usa fill() per riempire tutti gli spazi con 'vuoto'
- Stampa l'array della griglia
```

**English:**

```text
- Initialize an empty array of length 9
- Use fill() to set all spaces to 'empty'
- Print the grid array
```
