# Exercise 5

**Italiano:**

Data la lista VIP per una serata, verifica se l'utente X è presente nella lista VIP o se è nella lista standard.

**English:**

Given the VIP list for an evening, check if user X is on the VIP list or if they are on the standard list.

## Reasoning

**Italiano:**

```text
* Crea array vip e chiedi nome utente
* Se includes(nome utente):
  * Permetti l'ingresso
* Altrimenti:
  * Nega l'ingresso
```

**English:**

```text
* Create vip array and ask for username
* If includes(username):
  * Allow entry
* Otherwise:
  * Deny entry
```
