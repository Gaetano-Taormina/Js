# Exercise 3

**Italiano:**

Dati i posti liberi per carrozza [2, 0, 4, 7, 12, 8], chiedi all'utente il numero di passeggeri del gruppo e assegna loro la prima carrozza che ha capienza sufficiente, stampando il numero della carrozza scelta.

**English:**

Given the free seats per wagon [2, 0, 4, 7, 12, 8], ask the user for the number of passengers in the group and assign them the first wagon that has enough capacity, printing the chosen wagon number.

## Reasoning

**Italiano:**

```text
* Dichiara array posti
* Chiedi passeggeri
* Ciclo for per cercare la prima carrozza:
  * Se posti >= passeggeri:
    * Trova carrozza e interrompi
```

**English:**

```text
* Declare seats array
* Ask for passengers
* Loop to search for the first wagon:
  * If seats >= passengers:
    * Find wagon and break
```
