# Esercizio 25 / Exercise 25

**Italian:** Nella classifica di Serie A, prendi le squadre che giocheranno l'Europa League (dal 5° all'8° posto in classifica).

**English:** In the Serie A standings, take the teams that will play in the Europa League (from 5th to 8th place in the standings).

## Reasoning

**Italiano:**

```text
- Inizializza l'array della classifica
- Usa slice() per ottenere gli elementi dal 4 all'8 indice
- Stampa le squadre ottenute
```

**English:**

```text
- Initialize standings array
- Use slice() to get elements from index 4 to 8
- Print the extracted teams
```
