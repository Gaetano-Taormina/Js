# Esercizio 28 / Exercise 28

**Italian:** In un menu digitale, verifica se un ingrediente specifico è presente nella lista degli allergeni di un piatto.

**English:** In a digital menu, check if a specific ingredient is present in the allergen list of a dish.

## Reasoning

**Italiano:**

```text
- Crea l'array degli allergeni
- Usa includes() per verificare la presenza di Glutine
- Stampa il risultato booleano
```

**English:**

```text
- Create allergens array
- Use includes() to check for Gluten
- Print the boolean result
```
