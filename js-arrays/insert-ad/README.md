# Esercizio 29 / Exercise 29

**Italian:** Devi inserire una pubblicità dopo la terza notizia nel feed di un giornale online; inserisci l'elemento pubblicitario nella posizione corretta senza sovrascrivere nulla.

**English:** You need to insert an advertisement after the third news item in the feed of an online newspaper; insert the advertising element in the correct position without overwriting anything.

## Reasoning

**Italiano:**

```text
- Crea la lista dei feed
- Usa splice() all'indice 3 per inserire la Pubblicità senza eliminare nulla
- Stampa la lista
```

**English:**

```text
- Create feed list
- Use splice() at index 3 to insert Ad without deleting anything
- Print the list
```
