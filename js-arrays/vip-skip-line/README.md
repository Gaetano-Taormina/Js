# Exercise 19

**Italiano:**

Sei il buttafuori di un locale VIP: arriva un ospite importante che deve saltare la fila; inserisci il suo nome all'inizio della lista d'ingresso.

**English:**

You are the bouncer of a VIP club: an important guest arrives and needs to skip the line; insert their name at the beginning of the entry list.

## Reasoning

**Italiano:**

```text
* Dichiara array fila
* Usa unshift() per inserire all'inizio
* Stampa la lista aggiornata
```

**English:**

```text
* Declare line array
* Use unshift() to insert at the beginning
* Print the updated list
```
