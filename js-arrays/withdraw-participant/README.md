# Exercise 12

**Italiano:**

Un partecipante si ritira da una gara a metà percorso; rimuovi il suo nome dall'array della classifica mantenendo l'ordine degli altri.

**English:**

A participant withdraws from a race halfway through; remove their name from the ranking array keeping the order of the others.

## Reasoning

**Italiano:**

```text
* Dichiara classifica
* Trova l'indice del partecipante
* Usa splice per rimuoverlo
```

**English:**

```text
* Declare ranking
* Find the participant index
* Use splice to remove it
```
