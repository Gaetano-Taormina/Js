# Esercizio 24 / Exercise 24

**Italian:** In un magazzino, gli ultimi scatoloni arrivati sono i primi ad essere spediti. Il magazziniere spedisce un pacco, stampa la lista aggiornata.

**English:** In a warehouse, the last boxes to arrive are the first to be shipped. The warehouse worker ships a package, print the updated list.

## Reasoning

**Italiano:**

```text
- Inizializza l'array del magazzino
- Estrai l'ultimo elemento con pop()
- Stampa l'elemento spedito e l'array rimanente
```

**English:**

```text
- Initialize warehouse array
- Extract last element with pop()
- Print shipped element and remaining array
```
