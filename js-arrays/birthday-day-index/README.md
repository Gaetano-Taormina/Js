# Exercise 22

**Italiano:**

Trova la posizione del giorno della settimana in cui è caduto il tuo compleanno quest'anno.

**English:**

Find the position of the day of the week on which your birthday fell this year.

## Reasoning

**Italiano:**

```text
* Dichiara array giorni della settimana
* Usa indexOf() per trovare la posizione
```

**English:**

```text
* Declare days of the week array
* Use indexOf() to find the position
```
