# Exercise 8

**Italiano:**

Un ospedale ha 7 infermieri con questi turni: ["mattina", "pomeriggio", "notte", "mattina", "pomeriggio", "notte", "mattina"]. Conta quanti infermieri lavorano in ciascun turno.

**English:**

A hospital has 7 nurses with these shifts: ["morning", "afternoon", "night", "morning", "afternoon", "night", "morning"]. Count how many nurses work in each shift.

## Reasoning

**Italiano:**

```text
* Dichiara array turni
* Usa tre contatori
* Ciclo iterativo sui turni:
  * Incrementa il contatore corrispondente
```

**English:**

```text
* Declare shifts array
* Use three counters
* Loop over shifts:
  * Increment the corresponding counter
```
