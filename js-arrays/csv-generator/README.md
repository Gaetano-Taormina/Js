# Exercise 10

**Italiano:**

Devi generare un codice CSV da inviare al server; trasforma un array in una stringa con i valori separati da ",".

**English:**

You need to generate a CSV code to send to the server; transform an array into a string with comma-separated values.

## Reasoning

**Italiano:**

```text
* Dichiara array
* Usa il metodo join per separare da virgola
```

**English:**

```text
* Declare array
* Use the join method to separate by comma
```
