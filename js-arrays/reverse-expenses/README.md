# Exercise 13

**Italiano:**

Hai registrato le tue spese giornaliere in ordine cronologico; invertilo per vedere l'ultima spesa effettuata in cima alla lista.

**English:**

You have recorded your daily expenses in chronological order; reverse it to see the last expense made at the top of the list.

## Reasoning

**Italiano:**

```text
* Dichiara array spese
* Usa il metodo reverse() per invertirlo
* Stampa l'array invertito
```

**English:**

```text
* Declare expenses array
* Use the reverse() method to invert it
* Print the reversed array
```
