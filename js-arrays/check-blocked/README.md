# Exercise 14

**Italiano:**

Prima di aggiungere un amico sui social, controlla se il suo nickname è già presente nella tua lista di contatti bloccati.

**English:**

Before adding a friend on social media, check if their nickname is already on your blocked contacts list.

## Reasoning

**Italiano:**

```text
* Dichiara lista bloccati e nuovo amico
* Usa includes() per controllare
* Stampa il risultato
```

**English:**

```text
* Declare blocked list and new friend
* Use includes() to check
* Print the result
```
