# Exercise 4

**Italiano:**

Date le temperature settimanali, trova il valore massimo, il minimo e calcola la media aritmetica delle sette giornate.

**English:**

Given the weekly temperatures, find the maximum, minimum value and calculate the arithmetic mean of the seven days.

## Reasoning

**Italiano:**

```text
* Dichiara array temperature
* Inizializza min, max e somma
* Ciclo sulle temperature:
  * Aggiorna min e max
  * Calcola la somma
```

**English:**

```text
* Declare temperatures array
* Initialize min, max and sum
* Loop over temperatures:
  * Update min and max
  * Calculate sum
```
