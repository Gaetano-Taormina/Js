# Exercise 20

**Italiano:**

In un'app di ricette, hai gli ingredienti in un array; stampali in modo leggibile nella descrizione del piatto.

**English:**

In a recipe app, you have the ingredients in an array; print them legibly in the dish description.

## Reasoning

**Italiano:**

```text
* Dichiara array ingredienti
* Usa join() per formattarli come stringa leggibile
```

**English:**

```text
* Declare ingredients array
* Use join() to format them as a readable string
```
