# Exercise 15

**Italiano:**

Stai chattando con un tuo amico; quando arriva un nuovo messaggio, aggiungilo all'intera conversazione.

**English:**

You are chatting with a friend; when a new message arrives, add it to the entire conversation.

## Reasoning

**Italiano:**

```text
* Dichiara array conversazione
* Usa push() per aggiungere il nuovo messaggio alla fine
* Stampa la conversazione
```

**English:**

```text
* Declare conversation array
* Use push() to add the new message at the end
* Print the conversation
```
