# Esercizio 27 / Exercise 27

**Italian:** Stai creando una funzione "Undo" (Annulla); inverti l'ordine delle azioni compiute per poterle annullare nell'ordine corretto.

**English:** You are creating an "Undo" function; reverse the order of the actions performed so you can undo them in the correct order.

## Reasoning

**Italiano:**

```text
- Inizializza l'array delle azioni
- Usa reverse() per invertire l'ordine dell'array
- Stampa l'ordine invertito
```

**English:**

```text
- Initialize actions array
- Use reverse() to reverse the array order
- Print the reversed order
```
