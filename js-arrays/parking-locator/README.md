# Exercise 9

**Italiano:**

In un parcheggio numerato, l'utente cerca la sua auto; trova la posizione (indice) del suo numero di targa.

**English:**

In a numbered parking lot, the user looks for their car; find the position (index) of their license plate number.

## Reasoning

**Italiano:**

```text
* Usa l'array targhe
* Usa indexOf per trovare l'indice
```

**English:**

```text
* Use the plates array
* Use indexOf to find the index
```
