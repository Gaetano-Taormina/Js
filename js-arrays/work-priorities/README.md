# Esercizio 26 / Exercise 26

**Italian:** Hai una lista di priorità lavorative; il capo ti assegna un'emergenza da fare subito: inseriscila come primo elemento dell'array e stampa la lista.

**English:** You have a list of work priorities; the boss assigns you an emergency to do immediately: insert it as the first element of the array and print the list.

## Reasoning

**Italiano:**

```text
- Inizializza la lista delle priorità
- Usa unshift() per aggiungere l'emergenza all'inizio
- Stampa la lista aggiornata
```

**English:**

```text
- Initialize priorities list
- Use unshift() to add emergency at the beginning
- Print updated list
```
