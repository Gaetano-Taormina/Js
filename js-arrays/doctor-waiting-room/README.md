# Exercise 21

**Italiano:**

Gestisci una fila di pazienti in sala d'attesa; quando il dottore chiama il primo paziente, rimuovilo dalla lista.

**English:**

Manage a line of patients in the waiting room; when the doctor calls the first patient, remove them from the list.

## Reasoning

**Italiano:**

```text
* Usa shift() per estrarre il prossimo paziente
```

**English:**

```text
* Use shift() to extract the next patient
```
