# Exercise 18

**Italiano:**

Da un estratto conto bancario, estrai solo le ultime 10 transazioni effettuate.

**English:**

From a bank statement, extract only the last 10 transactions made.

## Reasoning

**Italiano:**

```text
* Dichiara array transazioni
* Usa slice() per prendere gli ultimi 10 elementi
```

**English:**

```text
* Declare transactions array
* Use slice() to take the last 10 elements
```
