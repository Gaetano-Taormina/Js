# Exercise 16

**Italiano:**

Devi inizializzare il report mensile per il mese successivo; imposta temporaneamente tutti i valori a "N/D".

**English:**

You need to initialize the monthly report for the following month; temporarily set all values to "N/A".

## Reasoning

**Italiano:**

```text
* Crea array vuoto per i giorni del mese
* Ciclo (30 volte):
  * Aggiungi "N/D"
```

**English:**

```text
* Create an empty array for the days of the month
* Loop (30 times):
  * Push "N/A"
```
