# Exercise 2

**Italiano:**

Dati i voti e i nomi degli studenti, stampa chi ha la sufficienza e chi no.

**English:**

Given the grades and names of the students, print who has a passing grade and who does not.

## Reasoning

**Italiano:**

```text
* Crea array nomi e array voti
* Ciclo for per iterare:
  * Se voto >= 6:
    * Stampa sufficiente
  * Altrimenti:
    * Stampa insufficiente
```

**English:**

```text
* Create names array and grades array
* Loop to iterate:
  * If grade >= 6:
    * Print passing
  * Otherwise:
    * Print failing
```
