# Exercise 2

**Italiano:**

Etichetta l'utente in base alla sua età. Sotto i 18 anni "Minorenne", "Giovane Adulto" tra 18 e 35, "Adulto" se tra 36 e 60, "Senior" oltre 60.

**English:**

Label the user based on their age. Under 18 "Minor", "Young Adult" between 18 and 35, "Adult" between 36 and 60, "Senior" over 60.

## Reasoning

**Italiano:**

```text
* Dichiara una funzione che accetta l'età
  * Se l'età è < 18:
    * restituisci "Minorenne"
  * Se è <= 35:
    * restituisci "Giovane Adulto"
  * Se è <= 60:
    * restituisci "Adulto"
  * Altrimenti:
    * restituisci "Senior"
```

**English:**

```text
* Declare a function that accepts age
  * If age is < 18:
    * return "Minor"
  * If it is <= 35:
    * return "Young Adult"
  * If it is <= 60:
    * return "Adult"
  * Otherwise:
    * return "Senior"
```
