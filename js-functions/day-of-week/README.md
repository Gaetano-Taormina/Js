# Exercise 1

**Italiano:**

Un calendario digitale deve mostrare il giorno esteso. Scrivi una funzione che riceve un numero da 1 a 7 e restituisce il nome del giorno corrispondente (1 = Lunedì, ..., 7 = Domenica). Se il numero non è valido, restituisce "Errore"

**English:**

A digital calendar must display the full day name. Write a function that takes a number from 1 to 7 and returns the corresponding day name (1 = Monday, ..., 7 = Sunday). If the number is invalid, it returns "Error".

## Reasoning

**Italiano:**

```text
* leggi il numero del giorno
* se il numero è compreso tra 1 e 7:
  * stampa il giorno corrispondente dell'array
* altrimenti:
  * stampa Errore
```

**English:**

```text
* read the day number
* if the number is between 1 and 7:
  * print the corresponding day from the array
* otherwise:
  * print Error
```
