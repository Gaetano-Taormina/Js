# Exercise Extra

**Italiano:**

* Un professore deve convertire un voto numerico in un giudizio. Scrivi una funzione che restituisca "Insufficiente", "Sufficiente" o "Ottimo" in base al punteggio.

**English:**

* A professor needs to convert a numerical grade into a judgment. Write a function that returns "Insufficient", "Sufficient" or "Excellent" based on the score.

## Reasoning

**Italiano:**

```text
* leggi voto
* dichiarare funzione che imposta un commento in base al voto:
  * se voto < 6:
    * stampa insufficiente
  * altrimenti se voto <= 8:
    * stampa buono
  * altrimenti:
    * stampa ottimo
```

**English:**

```text
* read grade
* declare function that sets a comment based on the grade:
  * if grade < 6:
    * print insufficient
  * else if grade <= 8:
    * print good
  * else:
    * print great
```
