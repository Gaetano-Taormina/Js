# Exercise

**Italian:** In aeroporto, i bagagli hanno categorie precise. Scrivi una funzione che riceve il peso in kg e restituisce "Bagaglio a mano" se >= 10kg, "Bagaglio da stiva" se >= 23kg, "Collo ingombrante" se superiore.

**English:** At the airport, luggage has precise categories. Write a function that receives the weight in kg and returns "Hand luggage" if >= 10kg, "Checked luggage" if >= 23kg, "Bulky item" if higher.

## Reasoning

**Italiano:**

```text
- Definisci la funzione che accetta il peso
- Se peso >= 23:
  - Ritorna 'Oggetto ingombrante'
- Se peso >= 10:
  - Ritorna 'Bagaglio da stiva'
- Altrimenti:
  - Ritorna 'Bagaglio a mano'
- Esegui e stampa il risultato
```

**English:**

```text
- Define function accepting weight
- If weight >= 23:
  - Return 'Bulky item'
- If weight >= 10:
  - Return 'Checked luggage'
- Else:
  - Return 'Hand luggage'
- Call and print result
```
